# WordLy — Руководство пользователя

## Для кого это руководство?

Этот документ предназначен для тех, кто впервые открывает приложение WordLy — как для обычных пользователей (учеников), так и для администраторов, добавляющих учебные материалы.

---

## Часть 1. Для учеников

### Как устроено обучение

Обучение в WordLy строится по простой схеме:

```
Курс → Уроки → Задания
```

Вы выбираете курс, внутри него проходите уроки по порядку, а каждый урок состоит из нескольких интерактивных заданий. За правильные ответы начисляются очки.

---

### Типы курсов

#### 📖 Словарный запас (Vocabulary)
Курсы на изучение отдельных слов и выражений. Подходят для новичков.
Вы будете учить новые слова, их перевод и написание.
*Пример: «Базовая лексика» — приветствия, семья, числа.*

#### 📝 Грамматика (Grammar)
Курсы на правила английского языка: времена глаголов, артикли, порядок слов в предложении.
Вы будете составлять и переводить предложения по грамматическим правилам.
*Пример: «Основы грамматики» — Present Simple, артикли a/an/the.*

#### 👂 Аудирование (Listening)
Курсы на понимание речи и разговорных фраз. Развивают навык восприятия английского языка.
Вы будете переводить фразы и выбирать правильные варианты из тех, что часто звучат в разговоре.
*Пример: «Слушай и понимай» — общие фразы, дни недели, время.*

#### 🗣️ Разговорная речь (Speaking)
Курсы на построение живых фраз и диалогов. Помогают научиться говорить по-английски в реальных ситуациях.
Вы будете составлять предложения из слов и переводить разговорные выражения.
*Пример: «Говори уверенно» — знакомство, распорядок дня.*

---

### Типы заданий

#### 🔤 Перевод слова (Word Translation)
Вам показывается одно слово на английском языке. Вы вводите его перевод на русский или казахский язык вручную.

**Как выполнять:**
Прочитайте слово → введите перевод в поле → нажмите «Проверить».

*Пример:*
> Слово: **Mother**
> Ваш ответ: *Мама*

---

#### ✍️ Перевод предложения (Sentence Translation)
Вам показывается целое предложение на английском языке. Вы переводите его письменно.

**Как выполнять:**
Прочитайте предложение → введите перевод → нажмите «Проверить».

*Пример:*
> Предложение: **The cat is on the table.**
> Ваш ответ: *Кошка на столе.*

---

#### 🔘 Выбор правильного ответа (Multiple Choice)
Вам задаётся вопрос, и предлагаются 4 варианта ответа. Нужно выбрать один правильный.

**Как выполнять:**
Прочитайте вопрос → нажмите на правильный вариант → нажмите «Проверить».

*Пример:*
> Вопрос: **Что значит «Goodbye»?**
> Варианты: До свидания / Привет / Спасибо / Пожалуйста
> Правильный ответ: *До свидания*

---

#### 🧩 Составление предложения (Sentence Building)
Вам даётся набор слов в перемешанном порядке. Нужно составить из них правильное предложение, нажимая на слова по очереди.

**Как выполнять:**
Слова внизу — это «кирпичики». Нажимайте на них, чтобы добавить в предложение сверху. Если добавили лишнее слово — нажмите на него в строке предложения, чтобы убрать обратно. Когда предложение готово — нажмите «Проверить».

*Пример:*
> Слова: **My / name / is / Alex / am / I**
> Правильное предложение: *My name is Alex*

---

### Прогресс и очки

- После каждого правильного ответа вы получаете **1 очко**.
- По завершении урока начисляется **бонус за урок** (зависит от курса).
- Все очки суммируются и отображаются в разделе **«Мой прогресс»**.
- Ваш текущий уровень (A1, A2 и т.д.) виден на главном экране.

---

### Раздел повторения

Если вы ответили на вопрос **неправильно**, задание сохраняется в разделе **«Повторение»**.

В разделе повторения вы снова увидите эти задания. Ответьте правильно — и ошибка будет удалена из списка.
Это помогает лучше запомнить трудные слова и выражения.

## Часть 2. Для администраторов

Чтобы попасть в административную панель, войдите в аккаунт администратора. На главном экране появится кнопка **«Admin Panel»**.

---

### Вкладка «Add Course» — Добавить курс

Здесь вы создаёте новый курс. Заполните все поля:

| Поле | Что вводить |
|---|---|
| English title | Название курса на английском |
| Russian title | Название курса на русском |
| Kazakh title | Название курса на казахском |
| English description | Краткое описание на английском |
| Russian description | Краткое описание на русском |
| Kazakh description | Краткое описание на казахском |
| Level | Уровень сложности: A1, A2, B1, B2, C1, C2 |
| Lessons | Примерное количество уроков в курсе |
| Points to unlock | Сколько очков нужно набрать, чтобы разблокировать курс (0 = открыт сразу) |
| Type | Тип курса: Vocabulary / Grammar / Listening / Speaking |

Нажмите **«Save Course»** — курс появится в списке на главном экране.

---

### Вкладка «Add Lesson» — Добавить урок

Здесь вы добавляете урок внутрь уже существующего курса.

| Поле | Что вводить |
|---|---|
| Course | Выберите курс из выпадающего списка |
| English / Russian / Kazakh title | Название урока на трёх языках |
| Order | Порядковый номер урока (1, 2, 3…) |
| Task count | Количество заданий в уроке |
| Points | Количество очков, которое получит ученик за прохождение урока |

Нажмите **«Save Lesson»** — урок появится внутри выбранного курса.

> ⚠️ Важно: добавляйте уроки **перед** заданиями, так как задание привязывается к конкретному уроку.

---

### Вкладка «Add Task» — Добавить задание

Здесь вы создаёте задание внутри урока. Сначала выберите курс и урок из выпадающих списков — они подтягиваются автоматически.

#### Общие поля для всех типов заданий

| Поле | Что вводить |
|---|---|
| Course | Выберите курс |
| Lesson | Выберите урок (список загружается после выбора курса) |
| Task type | Тип задания (см. ниже) |
| Question (English) | Вопрос или слово на английском языке |
| Answer (English) | Правильный ответ на английском |
| Answer (Russian) | Правильный ответ на русском |
| Answer (Kazakh) | Правильный ответ на казахском |

#### Дополнительные поля — зависят от типа задания

**Word Translation / Sentence Translation:**
Дополнительных полей нет. Заполните только вопрос и ответы на трёх языках.

**Multiple Choice:**
Появляется поле **«Options (comma separated)»** — введите 4 варианта ответа через запятую.
Один из вариантов должен совпадать с полем «Answer».

*Пример:* `До свидания, Привет, Спасибо, Пожалуйста`

**Sentence Building:**
Появляются два поля:
- **Words (comma separated)** — все слова для «кирпичиков», включая лишние, через запятую.
- **Correct sentence** — правильное предложение, которое должен собрать ученик.

*Пример слов:* `My, name, is, Alex, am, I`
*Правильное предложение:* `My name is Alex`

Нажмите **«Save Task»** — задание сохранится в Firestore и будет доступно при прохождении урока.

---

### Вкладка «Seed Data» — Пример данных

Нажмите **«Create Sample Data»** — система автоматически создаст 4 готовых курса с уроками и заданиями всех типов. Это удобно для первоначального наполнения приложения или тестирования.

Созданные курсы:
- 📖 **Basic Vocabulary** (A1) — базовые слова
- 📝 **English Grammar Basics** (A1) — грамматика
- 👂 **Listen & Understand** (A2) — аудирование
- 🗣️ **Speak English Confidently** (A2) — разговорная речь

---

### Порядок добавления контента

Всегда соблюдайте этот порядок:

```
1. Создайте курс (вкладка «Add Course»)
2. Создайте уроки для этого курса (вкладка «Add Lesson»)
3. Добавьте задания в каждый урок (вкладка «Add Task»)
```

Если нарушить порядок и попытаться добавить задание до создания урока — выпадающий список уроков будет пустым.
